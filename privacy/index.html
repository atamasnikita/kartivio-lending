3. Какие данные мы обрабатываем

3.1. Идентификационные данные

  • Telegram User ID, username и публичное имя (при входе через Telegram);
  • технический идентификатор аккаунта в Сервисе;
  • данные авторизации и токены сессии.

3.2. Данные использования

  • тексты запросов (промпты), выбранные настройки генерации;
  • история генераций, статусы задач, баланс кредитов;
  • IP-адрес, user-agent, дата и время запросов, технические логи.

3.3. Контентные данные

  • загруженные пользователем изображения для редактирования;
  • сгенерированные изображения как результат оказания услуги.

3.4. Платежные данные

  • статусы и история оплат, состав приобретенных пакетов кредитов;
  • идентификаторы платежей платежного провайдера;
  • контактные данные, переданные для выдачи электронного чека (при необходимости).

4. Цели обработки персональных данных

  • предоставление функциональности Сервиса (генерация и редактирование изображений);
  • авторизация пользователя и ведение аккаунта;
  • прием оплаты, учет кредитов, исполнение фискальных обязанностей;
  • обеспечение безопасности, предотвращение злоупотреблений, поддержка пользователей;
  • улучшение качества и стабильности работы Сервиса.

5. Правовые основания обработки

  • согласие субъекта персональных данных (п. 1 ч. 1 ст. 6 152-ФЗ);
  • исполнение договора с пользователем (п. 5 ч. 1 ст. 6 152-ФЗ);
  • исполнение обязанностей, предусмотренных законодательством РФ (в том числе 54-ФЗ).

Для работы Сервиса данные могут передаваться:

  • внешним AI-провайдерам для выполнения генерации/редактирования изображений;
  • провайдерам инфраструктуры (хостинг/хранение данных) в объеме, необходимом для оказания услуги.

7. Сроки хранения данных

  • данные аккаунта, платежей и операций по кредитам: в сроки, предусмотренные законом и учетом;
  • технические логи: в пределах срока, необходимого для безопасности и диагностики.

8. Права пользователя
