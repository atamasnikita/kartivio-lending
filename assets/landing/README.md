# Картинки для лендинга

## Куда класть файлы

- `assets/landing/hero/hero_01.jpg ... hero_08.jpg` — лента в hero-блоке.
- `assets/landing/how/before.jpg` — исходное фото в блоке «Как это работает».
- `assets/landing/how/after.jpg` — итоговый результат в блоке «Как это работает».
- `showcase.groups` в `manifest.json` — подборки реальных кадров для секции «Что можно получить».
- `how.results` в `manifest.json` — серия результатов для шага 4 в блоке «Как это работает».

## Что редактировать

Файл: `assets/landing/manifest.json`

Можно указывать:
- локальные пути (`/assets/landing/hero/hero_01.jpg`);
- внешние URL (например, S3/CDN `https://...`).

Новые секции тоже собираются из `manifest.json`:
- `showcase.groups[]` — табы и сетка примеров;
- `how.templates[]` — карточки шаблонов/сцен в шаге 2 (`title`, `badge`, `src`, `prompt`);
- `how.prompt` — текст примера промпта;
- `how.results[]` — мини-галерея готовых результатов.

## Рекомендации по формату

- Формат: `JPG` или `WEBP`.
- Hero-карточки: лучше портретные (3:4), `840x1120` или близко.
- Before/After: `1200x1600` или выше.
- Вес одного файла: желательно до `400 KB`.

## Быстрая проверка

1. Положи файлы в папки выше.
2. Обнови `manifest.json`, если имена/пути отличаются.
3. Открой `https://kartivio-ai.ru` (или локально) и проверь hero + секцию «Что можно получить» + блок «Как это работает».
